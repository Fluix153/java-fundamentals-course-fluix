# Código Fuente del Curso "Fundamentos de Java para Principiantes"

¡Bienvenido al repositorio de código de nuestro curso! Aquí encontrarás todos los ejemplos de código Java utilizados en las 6 lecciones de video.

## Estructura de Carpetas

* **/starter-files**: Contiene el código "inicial" para cada lección. Úsalo para programar junto con el video.
* **/completed-examples**: Contiene el código "final" de cada lección. Úsalo como referencia.
* **/scripts**: Contiene los guiones de narración detallados para cada video del curso.

## ¿Cómo usar esto?
Recomendamos usar [Replit.com](https://replit.com/).
1.  Ve a Replit.com y elige "Create Repl".
2.  Selecciona la plantilla "Java".
3.  Copia el código de nuestros archivos (ej. `Main.java`) en el editor de Replit y presiona "Run".
