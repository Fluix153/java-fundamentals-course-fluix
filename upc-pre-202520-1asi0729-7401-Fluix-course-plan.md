## Secuencia de la Lección

### Lección 1: ¿Qué es Java y la Programación? (5 minutos)
* [cite_start]**Descripción**: Aprende qué es la programación y por qué Java es un lenguaje tan popular para crear aplicaciones y juegos (¡como Minecraft!)[cite: 273].
* **Enlace**: [PEGA AQUÍ TU ENLACE DE YOUTUBE (Video 1)]
* [cite_start]**Conclusiones clave**: Java se usa para aplicaciones y juegos; los programas son instrucciones para computadoras[cite: 275].
* **Empieza a programar**: [PEGA AQUÍ TU ENLACE DE REPLIT (Lección 1)]

### Lección 2: Variables y tipos de datos (8 minutos)
* **Descripción**: Aprende a almacenar datos en Java usando variables. [cite_start]Escribirás tu primer "¡Hola Mundo!" y guardarás información en "cajas" llamadas variables[cite: 278].
* **Enlace**: [PEGA AQUÍ TU ENLACE DE YOUTUBE (Video 2)]
* **Conclusiones clave**: Usa `int` para números y `String` para texto. [cite_start]Declara variables como `int edad = 15;` o `String nombre = "Fido";`[cite: 280].
* **Práctica**: [PEGA AQUÍ TU ENLACE DE REPLIT (Lección 2)] ¡No es necesario registrarse!

### Lección 3: Estructuras de control: Decisiones y bucles (10 minutos)
* **Descripción**: ¡Dale un "cerebro" a tu programa! [cite_start]Aprende a tomar decisiones con `if-else` (si... entonces...) y a repetir tareas automáticamente con bucles `for`[cite: 283].
* **Enlace**: [PEGA AQUÍ TU ENLACE DE YOUTUBE (Video 3)]
* **Conclusiones clave**: `if (edad > 12) { ... }`; [cite_start]`for (int i = 0; i < 5; i++) { ... }` (Repite 5 veces)[cite: 285].
* **Práctica**: [PEGA AQUÍ TU ENLACE DE REPLIT (Lección 3)] ¡Inicio inmediato!

### Lección 4: Métodos, Entrada/Salida e Introducción a la POO (12 minutos)
* **Descripción**: Aprende a crear código reutilizable con "métodos". Haz tu programa interactivo leyendo la entrada del usuario con `Scanner`. [cite_start]Finalmente, veremos qué es una "Clase"[cite: 288].
* **Enlace**: [PEGA AQUÍ TU ENLACE DE YOUTUBE (Video 4)]
* **Conclusiones clave**: Crea métodos como `public static void saludar() { ... }`. Usa `Scanner` para leer texto. [cite_start]Una `Clase` es una plantilla (ej: `class Mascota { ... }`)[cite: 289].
* **Práctica**: [PEGA AQUÍ TU ENLACE DE REPLIT (Lección 4)] ¡No necesitas registrarte!

### Lección 5: Fundamentos de POO y programas sencillos (15 minutos)
* **Descripción**: ¡Vamos a construir con POO! Crearemos una clase `Mascota` con sus propios "atributos" (como nombre y edad) y "métodos" (como hacerSonido). [cite_start]Crearemos "objetos" desde esta plantilla[cite: 292].
* **Enlace**: [PEGA AQUÍ TU ENLACE DE YOUTUBE (Video 5)]
* **Conclusiones clave**: Aprende sobre Encapsulación (atributos `private`) y usa "getters" para leerlos. [cite_start]Crearemos un programa básico que usa un objeto `Mascota`[cite: 294].
* **Proyecto final**: [PEGA AQUÍ TU ENLACE DE REPLIT (Lección 5)] ¡Guarda y comparte!

### Lección 6: Consejos y próximos pasos (10 minutos)
* [cite_start]**Descripción**: Veremos los 5 errores más comunes que frustran a los nuevos programadores de Java (¡y cómo solucionarlos!) y te diremos qué puedes aprender después[cite: 297].
* **Enlace**: [PEGA AQUÍ TU ENLACE DE YOUTUBE (Video 6)]
* **Consejos clave**: ¡No olvides el punto y coma (`;`)! Revisa tus llaves (`{ }`). [cite_start]Sigue aprendiendo sobre Herencia y Polimorfismo[cite: 299].
* **Compartir tu trabajo**: [ENLACE A UN FORMULARIO OPCIONAL SI QUIERES CREAR UNO]
