# Fundamentos de Java para Principiantes

## Resumen del Curso
Este curso de 1 hora introduce a estudiantes de secundaria a los fundamentos de la programación con Java, incluyendo los conceptos básicos de la programación orientada a objetos (POO). ¡No requiere descargas! Solo abre tu navegador web.

* **Duración total**: ~60 minutos
* **Público objetivo**: Estudiantes de 12 a 17 años sin experiencia en programación
* **Prerrequisitos**: Ninguno
* **Herramientas necesarias**: ¡Solo tu navegador web! (Usaremos Replit.com)
* **Repositorio de código fuente**: [https://github.com/Fluix153/java-fundamentals-course-fluix](https://github.com/Fluix153/java-fundamentals-course-fluix)

## Secuencia de la Lección

### Lección 1: ¿Qué es Java y la Programación? (5 minutos)
* **Descripción**: Aprende qué es la programación y por qué Java es un lenguaje tan popular para crear aplicaciones y juegos (¡como Minecraft!).
* **Enlace**: [PEGA AQUÍ TU ENLACE DE YOUTUBE (Video 1)]
* **Conclusiones clave**: Java se usa para aplicaciones y juegos; los programas son instrucciones para computadoras.
* **Empieza a programar**: [PEGA AQUÍ TU ENLACE DE REPLIT (Lección 1)]

### Lección 2: Variables y tipos de datos (8 minutos)
* **Descripción**: Aprende a almacenar datos en Java usando variables. Escribirás tu primer "¡Hola Mundo!" y guardarás información en "cajas" llamadas variables.
* **Enlace**: [PEGA AQUÍ TU ENLACE DE YOUTUBE (Video 2)]
* **Conclusiones clave**: Usa `int` para números y `String` para texto. Declara variables como `int edad = 15;` o `String nombre = "Fido";`.
* **Práctica**: [PEGA AQUÍ TU ENLACE DE REPLIT (Lección 2)] ¡No es necesario registrarse!

### Lección 3: Estructuras de control: Decisiones y bucles (10 minutos)
* **Descripción**: ¡Dale un "cerebro" a tu programa! Aprende a tomar decisiones con `if-else` (si... entonces...) y a repetir tareas automáticamente con bucles `for`.
* **Enlace**: [PEGA AQUÍ TU ENLACE DE YOUTUBE (Video 3)]
* **Conclusiones clave**: `if (edad > 12) { ... }`; `for (int i = 0; i < 5; i++) { ... }` (Repite 5 veces).
* **Práctica**: [PEGA AQUÍ TU ENLACE DE REPLIT (Lección 3)] ¡Inicio inmediato!

### Lección 4: Métodos, Entrada/Salida e Introducción a la POO (12 minutos)
* **Descripción**: Aprende a crear código reutilizable con "métodos". Haz tu programa interactivo leyendo la entrada del usuario con `Scanner`. Finalmente, veremos qué es una "Clase".
* **Enlace**: [PEGA AQUÍ TU ENLACE DE YOUTUBE (Video 4)]
* **Conclusiones clave**: Crea métodos como `public static void saludar() { ... }`. Usa `Scanner` para leer texto. Una `Clase` es una plantilla (ej: `class Mascota { ... }`).
* **Práctica**: [PEGA AQUÍ TU ENLACE DE REPLIT (Lección 4)] ¡No necesitas registrarte!

### Lección 5: Fundamentos de POO y programas sencillos (15 minutos)
* **Descripción**: ¡Vamos a construir con POO! Crearemos una clase `Mascota` con sus propios "atributos" (como nombre y edad) y "métodos" (como hacerSonido). Crearemos "objetos" desde esta plantilla.
* **Enlace**: [PEGA AQUÍ TU ENLACE DE YOUTUBE (Video 5)]
* **Conclusiones clave**: Aprende sobre Encapsulación (atributos `private`) y usa "getters" para leerlos. Crearemos un programa básico que usa un objeto `Mascota`.
* **Proyecto final**: [PEGA AQUÍ TU ENLACE DE REPLIT (Lección 5)] ¡Guarda y comparte!

### Lección 6: Consejos y próximos pasos (10 minutos)
* **Descripción**: Veremos los 5 errores más comunes que frustran a los nuevos programadores de Java (¡y cómo solucionarlos!) y te diremos qué puedes aprender después.
* **Enlace**: [PEGA AQUÍ TU ENLACE DE YOUTUBE (Video 6)]
* **Consejos clave**: ¡No olvides el punto y coma (`;`)! Revisa tus llaves (`{ }`). Sigue aprendiendo sobre Herencia y Polimorfismo.
