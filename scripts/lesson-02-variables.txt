¡Vamos a analizar esto, porque parece intimidante!
`class Main { ... }`: Esto es nuestra "Clase" principal. Recuerda, el plano. Por ahora, todo nuestro código vivirá dentro de esta clase.
`public static void main(String[] args) { ... }`: Esta línea es un poco loca, lo sé. Por ahora, solo necesitas saber que este es el "punto de arranque" mágico. Cuando le damos a "Run", Java busca esta línea exacta para saber por dónde empezar. Nuestro código va *dentro* de estas llaves.

Ahora, la línea divertida:
`System.out.println("Hello world!");`
(Hablar sobre cada parte)
`System`... le decimos al sistema.
`.out`... que queremos usar la salida (la consola).
`.println`... que queremos "print line" (imprimir una línea).
`("Hello world!")`... y este es el texto que queremos imprimir. El texto siempre va entre comillas dobles.
Y lo más importante... **¡el punto y coma (`;`)!**
(Hacer zoom en el `;`)
En Java, CADA línea de instrucción, cada comando, debe terminar con un punto y coma. Es como el punto final de una oración. Si lo olvidas, el programa se rompe. ¡Lo veremos en un minuto!

Vamos a darle al botón verde "Run".
(Presionar "Run", esperar a que compile).
¡Miren! En la consola de la derecha, dice "Hello world!".
¡Felicidades! ¡Acabas de ejecutar tu primer programa en Java!

Vamos a cambiarlo. Borra "Hello world!" y pon "¡Hola, mi nombre es [Tu Nombre]!".

### (Continuación) Variables
Ahora, ¿qué pasa si queremos que el programa *recuerde* información?
Para eso, usamos **"variables"**. Una variable es una cajita en la memoria de la computadora, a la que le ponemos un nombre.

Hay muchos tipos de cajas, pero hoy veremos las 2 más importantes.
Primero, una caja para guardar texto. En Java, el texto se llama `String` (con 'S' mayúscula).
(Escribir)
`String nombre = "Fido";`
(Explicar) "Le decimos el *tipo* de caja (`String`), el *nombre* de la caja (`nombre`), el signo de igual para *asignar* el valor, y el valor (`"Fido"`), entre comillas y con punto y coma."

Segundo, una caja para números enteros (sin decimales). Se llama `int`.
(Escribir)
`int edad = 5;`
(Explicar) "Tipo: `int`. Nombre: `edad`. Valor: `5`. Nota que los números NO van entre comillas."

¿Y cómo las usamos? ¡Podemos imprimirlas!
(Escribir)
`System.out.println(nombre);`
`System.out.println(edad);`
(Presionar "Run")

Pero lo mejor es "concatenar" (unir) texto y variables. Usamos el símbolo `+`.
(Escribir)
`System.out.println("Mi perro se llama " + nombre);`
(Presionar "Run")
¡Miren! Java reemplazó la variable `nombre` con su valor, "Fido".
Vamos a hacerlo más complejo.
`System.out.println("Mi perro se llama " + nombre + " y tiene " + edad + " años.");`
(Presionar "Run")
¡Perfecto!

### Errores Comunes (Extenso)
¡Vamos a romper el código! ¡Es la mejor forma de aprender!
Error #1: ¡Olvidar el punto y coma!
(Borrar el `;` de la línea `int edad = 5;`)
(Presionar "Run")
¡BOOM! ¡Error! La consola se pone roja. Da miedo, ¿verdad? Pero no. Está siendo de ayuda.
Lee el error. Dice: "`Main.java:10: error: ';' expected`". ¡Te está diciendo exactamente qué pasa! "En el archivo Main.java, línea 10, esperaba un punto y coma".
(Poner el `;` de nuevo).

Error #2: Tipos incorrectos.
(Cambiar `int edad = 5;` a `int edad = "5";`)
"Le estoy diciendo a Java: 'guarda en esta caja de números... este texto'. Java no lo permite."
